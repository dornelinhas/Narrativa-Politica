Serviços

Edite os serviços diretamente no arquivo pages/servicos.html ou no assets/data/content.json. Os campos de texto dos cards de serviço foram projetados para fácil edição direta no HTML — procure pelas tags com classe service-big-title e service-big-subtitle.

Projetos & Métricas

Adicione novos projetos editando o arquivo pages/projetos.html. Cada projeto está num bloco .project-card. Copie um bloco existente e ajuste os dados. Os contadores de impacto são controlados pelo atributo data-count nas .stat-number.

Trilhas de Liderança

Edite os módulos em pages/trilhas.html. Cada módulo é um bloco .big-module. Para alterar preços, textos e tópicos, edite diretamente o HTML. Para adicionar um novo módulo, copie um bloco .big-module existente e ajuste o conteúdo.
